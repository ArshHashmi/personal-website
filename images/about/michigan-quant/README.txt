To add the next event as a new slide:
1) Duplicate the existing .about-slide block in index.html (inside .about-slides)
2) Point its 5 image paths to your new event folder images
3) Update data-kicker, data-title, data-meta, data-tags, and slide index text
4) Add one more dot button in .about-dots
